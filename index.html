📡 Fuentes de datos utilizadas:
• Dólar (oficial, blue, MEP, CCL, tarjeta, mayorista, cripto): DolarApi.com — API pública y gratuita, open-source (MIT). Agrega fuentes como Ámbito, La Nación, Infobae y datos oficiales del BCRA.
• Tasas (plazo fijo, BADLAR, TAMAR, cauciones): ArgentinaDatos.com — API pública argentina que centraliza datos del BCRA. | BCRA API oficial para variables principales.
• Riesgo País (EMBI+): ArgentinaDatos.com — Fuente: JP Morgan vía INDEC/BCRA.
• Bonos Soberanos (Globales, Bonares, Bopreales): data912.com — API argentina de mercado de capitales con datos de BYMA/MAE.
• ADRs, ETFs, Treasuries, Merval: Yahoo Finance vía proxy CORS público (query1.finance.yahoo.com) — Datos con retraso de 15 minutos.
• LECAPs y BONCAPs: argentinadatos.com — Filtra tickers S* (LECAPs) y T* (BONCAPs) del endpoint de bonos. TNA/TEA calculadas desde precio de mercado con liquidación T+1. Misma metodología que rendimientos.co (que usa data912, requiere servidor).
• Datos macroeconómicos: BCRA API oficial (reservas, base monetaria, tasas, tipo de cambio, UVA, préstamos, depósitos) + ArgentinaDatos.com (inflación IPC mensual/interanual, riesgo país EMBI+ vía JP Morgan).
• Rendimientos sectoriales YTD: Yahoo Finance — precios actuales vs cierre 31/12/2025. Calculado en tiempo real al abrir/actualizar.
• Rendimiento comparado LECAPs vs PF: TNA/TEA de LECAPs calculada desde precios de argentinadatos.com. Tasa PF desde BCRA API. BADLAR y TAMAR desde BCRA API.
• Nota: Los datos se actualizan al abrir la página y con el botón "Actualizar". Algunos datos pueden tener demora de 15–20 minutos según la fuente. Esta pantalla es de uso informativo personal.
